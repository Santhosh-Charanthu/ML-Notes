1. What is Linear Regression?

Linear Regression is a supervised machine learning algorithm used to predict continuous numerical values by learning the best-fit linear relationship between the input features and the target variable.

Examples

  • House Price Prediction
  • Salary Prediction
  • Temperature Prediction
  • Student Marks Prediction
  • Sales Forecasting

Interview Definition

Linear Regression is a supervised learning algorithm used to predict continuous numerical values by learning the best-fit linear relationship between the input features and the target variable.

2. Why is it Called "Linear" Regression?

It is called Linear Regression because it assumes there is an approximately linear (straight-line) relationship between the input features and the target variable.

For example:

Hours Studied Marks
1 10
2 20
3 30
4 40

The relationship can be represented by a straight line.

3. Real-World Applications

Linear Regression is used when the output is continuous.

Examples:

Problem Can Linear Regression Be Used?
House Price Prediction ✅ Yes
Salary Prediction ✅ Yes
Predict Temperature ✅ Yes
Student Marks Prediction ✅ Yes
Spam Detection ❌ No
Disease Prediction (Yes/No) ❌ No

4. Types of Linear Regression

Simple Linear Regression

Uses one input feature.

Example:

Hours Studied → Marks

Only one feature:

Hours Studied

Multiple Linear Regression

Uses multiple input features.

Example:

Predict House Price using:

  • Area
  • Bedrooms
  • Age of House

These multiple features are used together to predict the target.

6. Understanding coef_ and intercept_

coef_

Represents the slope of the regression line.

It tells us how much the predicted output changes when the input increases by one unit.

Example:

print(model.coef_)

Output:

[10.]

Meaning:

Every extra hour studied increases the predicted marks by 10.

intercept_

Represents the predicted output when the input is 0.

Example:

print(model.intercept_)

Output:

5.0

Meaning:

If a student studies 0 hours, the predicted marks are 5.

7. How Linear Regression Works

Suppose we have this dataset:

Hours Marks
1 10
2 20
3 30

Initially, the model doesn't know the relationship.

It starts with a random line, for example:

Marks = 4 × Hours + 8

Predictions:

Hours Actual Predicted
1 10 12
2 20 16
3 30 20

The predictions contain errors.

The model calculates these errors and repeatedly adjusts the slope and intercept until it finds the line that minimizes the prediction error.

8. Cost Function (Concept)

The Cost Function measures how far the model's predictions are from the actual values.

  • High Cost → Poor predictions
  • Low Cost → Better predictions

The objective of Linear Regression is to minimize the cost function.

Interview Note: For campus placements, understanding the concept is sufficient. You don't need to derive the formula.

9. Model Training

Training is done using:

model.fit(X, y)

What happens internally?

  • The model analyzes the training data.
  • It learns the relationship between the features and the target.
  • It finds the best values for the slope (coef_) and intercept (intercept_) that minimize prediction error.

10. Prediction

After training:

model.predict([[6]])

The model uses the learned equation to predict the output for unseen data.

11. Assumptions of Linear Regression

  1. Linear Relationship The relationship between input and output should be approximately linear.

  2. Independent Observations One data point should not depend on another.

  3. No Extreme Outliers Large outliers can distort the regression line.

  4. Low Multicollinearity Input features should not be highly correlated with each other.

  5. Normally Distributed Errors The prediction errors should be approximately normally distributed.

12. Advantages

  • Easy to understand and implement.
  • Fast to train.
  • Highly interpretable.
  • Works well for linear relationships.
  • Baseline algorithm for many regression problems.

13. Limitations

  • Assumes a linear relationship.
  • Sensitive to outliers.
  • Performs poorly on complex non-linear data.
  • Not suitable for classification problems.

14. Scikit-Learn Implementation

from sklearn.linear_model import LinearRegression
import numpy as np

X = np.array([[2], [4], [6], [8], [10]])
y = np.array([20, 40, 60, 80, 100])

model = LinearRegression()

model.fit(X, y)

print("Coefficient:", model.coef_)
print("Intercept:", model.intercept_)

prediction = model.predict([[12]])

print("Prediction:", prediction)

🎯 Top 10 Interview Questions with Answers

1. What is Linear Regression?

Answer:

Linear Regression is a supervised machine learning algorithm used to predict continuous numerical values by learning the best-fit linear relationship between the input features and the target variable.

2. Why is it called Linear Regression?

Answer:

It is called Linear Regression because it assumes there is an approximately linear (straight-line) relationship between the input features and the target variable and models this relationship using the best-fit line.

3. When should we use Linear Regression?

Answer:

Linear Regression should be used when the target variable is continuous and there is an approximately linear relationship between the input features and the target.

Examples include predicting house prices, salaries, and temperatures.

4. What is the difference between Simple and Multiple Linear Regression?

Answer:

Simple Linear Regression uses one input feature to predict the target variable, whereas Multiple Linear Regression uses two or more input features.

5. What does model.fit(X, y) do?

Answer:

model.fit(X, y) trains the model using the training data. It learns the best-fit relationship by finding the optimal values of the coefficient (coef_) and intercept (intercept_) that minimize prediction error.

6. What is coef_?

Answer:

coef_ is the slope of the regression line. It indicates how much the predicted output changes for a one-unit increase in the input feature.

7. What is intercept_?

Answer:

intercept_ is the predicted value of the target variable when all input features are zero.

8. What are the assumptions of Linear Regression?

Answer:

  • Linear relationship
  • Independent observations
  • No extreme outliers
  • Low multicollinearity
  • Normally distributed errors

9. Why can't Linear Regression be used for classification?

Answer:

Linear Regression predicts continuous numerical values and can produce outputs outside the valid range of class labels (for example, values less than 0 or greater than 1). Classification problems require predicting categories, so algorithms like Logistic Regression are more appropriate.

10. What are the advantages and limitations of Linear Regression?

Answer:

Advantages:

  • Simple and easy to interpret.
  • Fast to train.
  • Works well for linear relationships.

Limitations:

  • Assumes a linear relationship.
  • Sensitive to outliers.
  • Not suitable for non-linear data or classification tasks.
