1. What is Logistic Regression?

Logistic Regression is a supervised machine learning algorithm used for classification problems.

Unlike Linear Regression, which predicts continuous values, Logistic Regression predicts categories (classes).

Examples

Problem Output
Spam Detection Spam / Not Spam
Disease Prediction Yes / No
Student Pass Prediction Pass / Fail
Loan Approval Approved / Rejected
Fraud Detection Fraud / Genuine

Notice that all outputs are categories, not numbers like salary or house price.

2. Why Can't We Use Linear Regression for Classification?

Let's say we're predicting whether a student passes.

We assign:

  • Pass = 1
  • Fail = 0

Using Linear Regression, the model might predict:

-0.5
1.7
2.3

These are not valid probabilities or class labels.

For classification, we need outputs between:

0 and 1

That's why Linear Regression isn't suitable for classification.

3. Why is it Called "Regression"?

This confuses many beginners.

Although Logistic Regression is used for classification, it performs regression internally to estimate a score, then converts that score into a probability using the Sigmoid Function.

So the name "Regression" comes from the mathematical model it uses internally, while the final output is used for classification.

This is a very common interview question.

4. The Sigmoid Function

The Sigmoid Function converts any real number into a value between 0 and 1.

It is defined as:

You do not need to memorize the formula for campus placements.

Just remember:

  • Input: Any real number (positive or negative)
  • Output: A probability between 0 and 1

Examples

Input Sigmoid Output
-10 ~0.00
0 0.50
2 ~0.88
10 ~1.00

5. How Does Logistic Regression Predict?

Suppose we're predicting whether a student passes.

After applying the Sigmoid Function, the model outputs:

0.85

Interpretation:

There is an 85% probability that the student will pass.

Now we apply a decision threshold (usually 0.5).

Probability Prediction
≥ 0.5 Class 1 (Pass)
< 0.5 Class 0 (Fail)

Example:

  • 0.82 → Pass
  • 0.73 → Pass
  • 0.40 → Fail
  • 0.12 → Fail

6. Linear Regression vs Logistic Regression

Linear Regression Logistic Regression
Regression algorithm Classification algorithm
Predicts continuous values Predicts categories
Output can be any real number Output is a probability (0 to 1)
Best-fit straight line Sigmoid curve
Used for house price, salary Used for spam detection, disease prediction

This comparison is asked in almost every fresher interview.

7. Training the Model

from sklearn.linear_model import LogisticRegression

model = LogisticRegression()

model.fit(X_train, y_train)

Just like Linear Regression:

  • fit() trains the model.
  • It learns the relationship between features and labels.

8. Making Predictions

predictions = model.predict(X_test)

This returns class labels such as:

[1, 0, 1, 1, 0]

If you want the probabilities instead:

probabilities = model.predict_proba(X_test)

Example output:

[[0.90, 0.10],
 [0.25, 0.75]]

For the first sample:

  • 90% probability of Class 0
  • 10% probability of Class 1

9. Real-World Example

Suppose we want to predict whether a student passes based on hours studied.

Hours Studied Pass
1 0
2 0
3 0
4 1
5 1
6 1

Here:

  • Feature: Hours Studied
  • Label: Pass (0/1)

Logistic Regression learns from this data and predicts the probability of passing for new students.

10. Advantages

  • Simple and fast.
  • Easy to interpret.
  • Works well for binary classification.
  • Outputs probabilities.
  • Performs well on linearly separable data.

11. Limitations

  • Assumes a linear relationship between features and the log-odds of the target.
  • Not suitable for highly complex non-linear relationships.
  • Sensitive to outliers and irrelevant features.

For placements, it's enough to remember that Logistic Regression works best when the relationship is reasonably simple.

💻 Hands-On Activity

from sklearn.linear_model import LogisticRegression
from sklearn.datasets import load_iris

iris = load_iris()

X = iris.data
y = iris.target

model = LogisticRegression(max_iter=200)

model.fit(X, y)

print(model.predict([X[0]]))
print(model.predict_proba([X[0]]))
